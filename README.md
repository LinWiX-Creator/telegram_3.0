# Telegram 3.0 🚀

Современный MVP мессенджера (аналог Telegram).

## 🔥 Функции
- Реалтайм чат (Socket.io)
- Пользователи
- Личные сообщения
- Масштабируемая архитектура

## 🛠 Стек
- Node.js
- Express
- Socket.io

## 🚀 Запуск backend
```bash
cd backend
npm install
node index.js
```
